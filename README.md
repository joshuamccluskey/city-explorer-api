## Getting Started

- Clone Down GitHub Repo [GitHub Repo](https://github.com/joshuamccluskey/city-explorer-api)
- `cd city-explorer-api`
- `npm init`
  - endpoint: `server.js`
- `npm install express`
- `npm install dotenv`
- `npm install cors`
- `cp (Configs docs path) .(currentdirectory)`
- `npm -g i nodemon`
- `nodemon`
